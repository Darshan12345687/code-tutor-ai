# SEMO Logo Setup Instructions

## Current Logo Path
The application is configured to look for the SEMO logo at:
- **Primary**: `/assets/logos/semo logo.png` (with space)
- **Fallback 1**: `/assets/logos/semo-logo.png` (with hyphen)
- **Fallback 2**: `/assets/logos/semo-logo.svg` (SVG format)

## To Add Your SEMO Logo

1. Place your SEMO logo PNG file in:
   ```
   frontend/public/assets/logos/semo logo.png
   ```
   
   OR

   ```
   frontend/public/assets/logos/semo-logo.png
   ```

2. The logo will automatically appear in:
   - Header (main navigation)
   - Login page
   - Signup page

## File Requirements
- **Format**: PNG (preferred) or SVG
- **Size**: Recommended 200x60px or similar aspect ratio
- **Background**: Transparent preferred
- **Color**: Should match SEMO brand colors (#C8102E red)
